# Getting Started

## Prerequisites

- [Claude Code](https://docs.anthropic.com/en/docs/claude-code) installed
- This repo cloned locally

## Install

```
/council:setup
```

The setup skill:
1. Detects your shell (zsh/bash/fish)
2. Explains what will change and asks for confirmation
3. Asks you to pick an alias name
4. Checks the name isn't taken
5. Writes two lines to your shell config:
   - A shell alias so you can launch with the council plugin loaded from any directory
   - `CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS=1` — required for parallel wave execution
6. Tells you to `source` your config

After setup, launch from anywhere with your alias.

## Quick Start

Evaluate a project with the default council (4 data + 5 eval + 1 synth = 10 agents):

```
/council:evaluate Aave DAO
```

Output: `council-out/aave-dao/REPORT.md`

## Design a Custom Council

```
/council:design DeFi protocols
```

Two phases:

**Phase A — Planning**
1. Conversational domain discovery (what matters, what data exists)
2. Roster review by wave (add/remove agents)
3. Per-agent config (sources for data agents, 5 dimensions for eval agents)
4. Architecture confirmation → plan written via plan mode

**Phase B — Implementation** (after you approve)
1. Research each new agent's domain (parallel)
2. Generate agent definitions (parallel)
3. Generate documentation (parallel)
4. Remove deselected defaults, update README

## Add or Remove Agents

```
/council:add-agent                    # interactive
/council:add-agent eval governance    # direct
/council:remove-agent eval-financial  # with impact preview
```

## Agent Naming

| Prefix | Wave | Tools | Scores |
|--------|------|-------|--------|
| `data-*` | 1 | WebSearch, WebFetch | — |
| `eval-*` | 2 | Read (data/ only) | 5 dimensions, 1-10 |
| `synth-*` | 3 | Read (eval/ only) | — |

Filename prefix = wave assignment. No config needed.

## Manual Agent Creation

```bash
cp agents/eval-technical.md agents/eval-governance.md
# Edit: frontmatter (name, description), 5 scoring dimensions, calibration table
# Auto-discovered on next /council:evaluate
```

## Directory Layout

```
.claude-plugin/plugin.json   ← name: "council" (sets /council: prefix)
agents/data-*.md             ← Wave 1
agents/eval-*.md             ← Wave 2
agents/synth-*.md            ← Wave 3
skills/*/SKILL.md            ← Skill definitions
docs/skill-*.md              ← Skill documentation
research/                        ← Domain research (generated by design/add-agent)
council-out/                     ← Evaluation output
└── {slug}/data/ eval/ REPORT.md
```
